Overview

@suluk/scalar — render an OpenAPI v4 "Suluk" document with Scalar API Reference.

Scalar consumes OpenAPI 3.x, so we project the v4 doc down to 3.1 (via @suluk/openapi-compat) and embed the result inline. The page is self-contained: it loads the Scalar standalone bundle from a CDN and initializes it with the spec via Scalar.createApiReference('#app', { content }). CANDIDATE tooling.
