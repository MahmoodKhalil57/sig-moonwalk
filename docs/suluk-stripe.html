Overview

@suluk/stripe — first-class Stripe behind a swappable PaymentProvider. Usage-based billing via the modern Billing Meters API (meters + meter events + metered prices), customers, subscriptions, and webhooks — plus a bridge that turns @suluk/cost events into the usage you bill on. Stripe is the reference processor; the PaymentProvider interface is the swap point for the others that follow it. CANDIDATE tooling.
