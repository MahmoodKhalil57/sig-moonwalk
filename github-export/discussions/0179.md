### Comment 1 — @darrelmiller · 2025-03-25 17:00Z

We discussed this example including the topic of whether server URL was part of the signature, and whether signatures in resources could only add to signature at the root or could override.

```yaml
openapi: 4.0.0
servers:
  - url: https://{tenant}.com/api
    variables:
      tenant:
        default: myshop
signature:
  - serverUrl # Implicit
  - uriTemplate
  - method
resources:
    fooAsCSV:
        signature:
            - headers: {"format"}
        uriTemplate: /foo/{id}
        operations:
            - method: GET
    fooAsJson:
        signature:
            - headers: {"format"}
        uriTemplate: /foo/{id}
        operations:
            - method: GET
    specialFoo:
        uriTemplate: /foo/special
        operations:
```
