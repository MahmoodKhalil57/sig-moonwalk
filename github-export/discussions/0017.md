## Body

What if instead of overlays, you could define the merge rules like layers in Photoshop (a kind of makefile, if you will)? This might be an optional `manifest` or `layers` in the `info` object, and it defines the tasks to be executed. A fully resolved document would remove those instructions for a self-contained document.
